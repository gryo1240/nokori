# のこりレシピ

冷蔵庫にある食材をチェックするだけで、今日作れる料理が見つかる無料のWebアプリ(PWA)。

**アプリ本体: https://gryo1240.github.io/nokori/**

- **食材をタップしてチェック**: 肉・魚/野菜・きのこ/卵・豆腐・乳製品/麺・米・パン/その他に分類された約60品目からタップで選択
- **作れる順に自動で並び替え**: 内蔵36品のレシピを「材料がすべてそろっている」→「あと1品」→「あと2品」の順に自動ソート。足りない食材も表示
- **調理時間で絞り込み**: 10分/15分/20分/30分以内で候補を絞れる
- **お気に入り**: よく作る料理に☆をつけて次回すぐ呼び出せる
- **足りない食材をコピー**: レシピ詳細から、買い足す分だけをテキストでコピーできる
- 何も選んでいない状態では全36品をブラウズ表示。分量は2人分の目安、調味料は常備している前提で材料には含めない
- 選んだ食材・お気に入りはすべて端末内(localStorage)に保存。外部送信・登録不要

姉妹アプリ「週の献立プランナー」( https://github.com/gryo1240/kondate )と同じ36品のレシピデータを使用。あちらは1週間分の献立を自動で組む用途、こちらは「今ある食材で今日何を作るか」を即決める用途です。

## 構成

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体(HTML/CSS/JSすべて1ファイル。レシピデータ36品を内蔵) |
| `manifest.webmanifest` / `sw.js` | PWA化(インストール・オフライン対応) |
| `icon-*.png` / `apple-touch-icon.png` | アプリアイコン |
| `docs/screenshot-*.png` | 紹介用スクリーンショット |

`main` ブランチに push すると GitHub Pages に自動デプロイされます。

ブログ等での紹介記事を書く場合は [HANDOVER.md](HANDOVER.md) を参照してください。
